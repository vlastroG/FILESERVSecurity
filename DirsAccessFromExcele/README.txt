# FILESERVSecurity

Простое и полезное приложение для Windows 10 для назначения прав доступа для Пользователей
к папкам файл-сервера Windows 10 (проекта в CDE) по маске, составленной в Excel файле
и обнуления доступа для пользователя ко всем папкам проекта.
Перед работой с программой необходимо ознакомиться с правилами составления excel файла.

Правила составления excel файла:
	1.Пути к подпапкам проекта должны быть относительными,
		относительно текущего корневого каталога проекта.
	2.Пути в excel должны строго совпадать с путями к существующим папкам.
	3.Пути писать в порядке иерархии папок в ячейках столюца A, начиная с 4 строчки.
	4.Имена пользователей писать строго в соответствии с именем пользователя windows
		в формате 'ivanov.ii'
	5.Имена писать в 3 строке начиная со столюца B.
	6.Если требуется в одной ячейке написать несколько имен пользователей,
		то в качестве разделителя использовать ';' и после нажимать Alt+Enter для переноса строки
	7.Матрица доступа должна составляться в соответствии с правилами:
		7.1. Если в текущей папке все ее подпапки и файлы имеют одинаковые права доступа,
			 то необходимо у этой папки назначить необходимые права доступа
			 и применить их "для этой папки, ее подпапок и файлов".
			 Для вложенных папок оставлять пустые ячейки.
		7.2. Если в текущей папке ее подпапки имеют права доступа отличные от текущей,
			 то необходимо назначить необходимые права доступа "только для текущей папки".
			 Для каждой вложенной папки руководствоваться снова правилом 7.
			 (либо применять права "только для текущей папки" и для каждой вложенной
			 снова либо "только для текущей",
			 либо "для текущей папки ее подпапок и файлов" и ее вложенные папки не рассматривать.)
	8.Реализовано 4 правила доступа к папкам:
		8.1. Нет доступа													в ячейке писать "-".
		8.2. Доступ на чтение только текущей папки							в ячейке писать "Ч".
			 Ставится "Разрешить" напротив  "Чтение и выполнение"
											"Список содержимого папки"
											"Чтение".
		8.3. Доступ на чтение текущей папки, ее подпапок и файлов			в ячейке писать "Ч!".
			 Ставится "Разрешить" напротив  "Чтение и выполнение"
											"Список содержимого папки"
											"Чтение".
		8.4. Доступ на редактирование текущей папки, ее подпапок и файлов	в ячейке писать "Р".
			 Ставится "Разрешить" напротив  "Изменение"
											"Чтение и выполнение"
											"Список содержимого папки"
											"Чтение"
											"Запись"
		*	 Все символы писать кириллицей.
	9.Если ячейка пустая, то либо нет прав доступа,
		либо права доступа наследуются от родительской папки.
